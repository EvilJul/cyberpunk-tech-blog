前端组件设计

基于React + Tailwind CSS的博客组件体系

组件树
<App>
← src/App.jsx，单页无路由
├─ [背景层] (fixed, z-0~2)
├─ <PixelSnow /> WebGL像素雪花粒子
├─ <SideRays /> WebGL角落光线
└─ <div.grid-bg /> CSS网格叠加
├─ [交互层] (fixed, z-50)
├─ <SplashCursor /> WebGL流体墨水飞溅
├─ <Header /> 毛玻璃导航栏
└─ <SearchBar /> 搜索组件
├─ [主内容] (relative, z-10)
├─ <Hero /> 博客标题区域
├─ <ArticleList /> 文章列表
│ ├─ <ArticleCard /> 文章卡片
│ └─ <TagFilter /> 标签筛选
├─ <ArticleDetail /> 文章详情
│ ├─ <CodeBlock /> 代码高亮
│ └─ <CommentSection /> 评论区
└─ <Footer /> 页脚
└─ [管理后台] (独立页面)
└─ <AdminPanel /> 管理后台
├─ <ArticleEditor /> 文章编辑器
├─ <CommentManager /> 评论管理
└─ <MediaUpload /> 图片上传

核心组件详解

Header组件
• 固定顶栏，滚动50px触发毛玻璃效果
• 包含：Logo、导航菜单、搜索图标
• 响应式：移动端汉堡菜单
ArticleCard组件
• 文章卡片展示：标题、摘要、标签、日期
• 悬停效果：边框发光、轻微上浮
• 点击跳转到文章详情
CodeBlock组件
• 代码高亮：使用highlight.js或Prism
• 暗金主题代码样式
• 复制按钮、行号显示
CommentSection组件
• 评论列表展示
• 评论表单：昵称、邮箱、内容
• 回复功能、嵌套评论
