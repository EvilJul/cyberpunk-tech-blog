## 🐛 已修复的问题

### 问题 #1: SPA Fallback 拦截 API 路由 ✅
**症状**: `/api/stats` 返回 HTML 而非 JSON  
**原因**: Express 5 路由顺序错误  
**修复**: 将 SPA fallback 移到中间件链末尾  
**验证**: 所有 API 端点正常返回 JSON ✅

### 问题 #2: Express 5 通配符语法错误 ✅
**症状**: `app.get('*')` 导致 PathError  
**原因**: Express 5 不支持 `*` 通配符  
**修复**: 改用 `app.use()` 中间件  
**验证**: 服务器正常启动 ✅
